# 01 - Criação da Máquina Virtual

No VMware Workstation Pro, clique em:

```text
File > New Virtual Machine
```

Selecione:

```text
Typical (recommended)
```

Escolha a ISO do Windows Server 2025.

---

## Sistema operacional convidado

Caso o VMware não tenha uma opção específica para Windows Server 2025, selecione:

```text
Microsoft Windows Server 2022 (64-bit)
```

Essa escolha serve apenas como perfil de compatibilidade inicial da VM.

---

## Nome da máquina virtual

Nome sugerido:

```text
Windows-Server-2025-Lab
```

---

## Disco virtual

Sugestão:

```text
60 GB
```

Para laboratório, é possível armazenar o disco como arquivo único ou dividido em múltiplos arquivos.
