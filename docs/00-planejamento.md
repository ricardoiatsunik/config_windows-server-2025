# 00 - Planejamento do Laboratório

Antes de criar a máquina virtual, é importante definir o objetivo do ambiente.

Neste laboratório, o objetivo é criar uma instalação limpa do Windows Server 2025 em VMware Workstation Pro, deixando a VM pronta para próximos estudos.

---

## Objetivo

Preparar uma máquina virtual base com Windows Server 2025.

Essa VM poderá ser reaproveitada para laboratórios futuros, como:

- Active Directory
- DNS
- DHCP
- GPO
- File Server
- Hardening
- Administração remota
- Estudos de segurança em Windows Server

---

## Recursos recomendados

| Recurso | Valor sugerido |
|---|---|
| CPU | 2 núcleos |
| RAM | 4 GB ou mais |
| Disco | 60 GB ou mais |
| Rede | NAT no primeiro momento |
| Firmware | UEFI |
| Instalação | Desktop Experience |

---

## Observação

O objetivo não é montar um ambiente de produção, mas sim uma base funcional para laboratório.

Em produção, a definição de CPU, RAM, disco, backup, segurança, licenciamento e rede dependeria da função real do servidor.
